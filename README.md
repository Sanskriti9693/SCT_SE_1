# Sudoku Solver using C++

##  Overview
This project is a Sudoku Solver developed using C++ and the Backtracking Algorithm.  
The program automatically solves an incomplete Sudoku puzzle while following all Sudoku rules.

---

##  Features
- Solves valid Sudoku puzzles automatically
- Uses Backtracking and Recursion
- Checks row, column, and 3×3 subgrid constraints
- Beginner-friendly implementation
- Efficient puzzle-solving logic

---

##  Technologies Used
- C++
- Recursion
- Backtracking Algorithm
- Data Structures & Algorithms
